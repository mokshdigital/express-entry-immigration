# Express Entry Immigration Services

Modern, headless CMS-powered immigration consulting website.

## Tech Stack
- **Frontend:** Next.js 14 + Tailwind CSS + shadcn/ui
- **Backend:** WordPress Headless CMS
- **Hosting:** Vercel (frontend) + Hostinger (WordPress)

## Project Structure
- `/frontend` - Next.js application
- `/docs` - Project documentation
- `/wordpress-setup` - WordPress configuration guides

## Getting Started
See `/docs/CONTEXT.md` for current project status.

## Development
```bash
cd frontend
npm install
npm run dev
```

## Deployment
Automatic deployment via Vercel on push to main branch.
